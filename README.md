# MTProxy Manager

Web-based management panel for MTProto proxy (mtg) containers. Create, start, stop and delete Telegram proxies with a user-friendly dashboard.

## Features

- Multi-user authentication (registration, login, JWT)
- Per-user proxy limits (configurable by admin)
- Admin panel: manage users, roles, limits, and all proxies
- Automatic port allocation from a configurable range
- Docker SDK integration (no shell scripts)
- Dark-themed responsive UI
